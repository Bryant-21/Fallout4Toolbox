## Fallout4Toolbox

Just my collection of python scripts to help speed up modding.

### DDS Files

To work with DDS you need texconv.exe. It's free from Microsoft. You then need to open the settings and add its location.

https://github.com/microsoft/DirectXTex/releases

### Requirements

.NET needed for MIP Flooding. [https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48](https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime)

```
pip install -r requirements.txt
```

### Run
```
python -m src.FalloutToolbox
```
