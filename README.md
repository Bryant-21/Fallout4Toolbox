## Fallout4Toolbox

Just my collection of python scripts to help speed up modding.

### Build Locally

```
pip install -r requirements.txt
```

### Run
```
python -m src.FalloutToolbox
```


### Implements AGPL Libraries

- https://github.com/BadDogSkyrim/PyNifly
